#+title: EmacsListen
Framework for voice driven computer interaction with Emacs.

* Installation
This repository contains the core for evaluating commands from a
speech recognition system.  To set this up, add the following to your
configuration:

#+begin_src elisp
  (use-package emacslisten
    :vc (:url "https://github.com/ErikPrantare/EmacsListen"
              :branch "master"
              :rev :newest))
#+end_src

To receive commands, Emacs must run as a server.  This can be achieved
by adding ~(server-start)~ to your configuration.

Commands are invoked in EmacsListen by calling ~emacslisten-evaluate~
with the lisp form that should be evaluated.

** Phony
To be able to define voice commands from Emacs lisp, install [[https://github.com/ErikPrantare/phony.el][phony.el]].
That repository also contains information on how to set up a speech
recognition engine.
